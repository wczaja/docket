# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. The full text at that link is authoritative; the
summary below is a convenience, not a replacement.

## In short

- Be welcoming, patient, and respectful in issues, PRs, reviews,
  discussions, and any other project space.
- Critique code and ideas, never people. Disagreement is fine;
  disrespect is not.
- Harassment, discrimination, personal attacks, trolling, and publishing
  others' private information are not tolerated, in project spaces or
  when officially representing the project elsewhere.
- Assume good faith from newcomers; remember that contributors come from
  many backgrounds and time zones.

## Scope

This code of conduct applies in all project spaces (the GitHub
repository, its issues, pull requests, and discussions) and when an
individual is officially representing the project in public spaces.

## Enforcement

Report unacceptable behavior privately to the maintainer at
**william.czaja@gmail.com**. Reports are reviewed promptly and handled
confidentially; the reporter's privacy and safety come first.

Maintainers may edit or remove comments, commits, issues, and PRs that
violate this code of conduct, and may temporarily or permanently ban any
contributor for behavior they deem inappropriate, following the
enforcement guidelines in Contributor Covenant 2.1.
